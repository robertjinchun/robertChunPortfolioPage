    • Type Alot

    Type Alot is a typing game that was created with JavaScript using JQuery Library.

    The objective of the game is to type the letters that appear as fast as you can to achieve the highest score

    Features:

    Use event handlers to achieve inputs from users keyboard

    Use the rest feature to split a four letter word in to individual characters

    Convert an individual character inputed by the user and individual character of each letter in the word in to ascii numbers and compare the values with one another

    Real time scoring system and timer incoperated
